This page outlines how the information in the XML files have been changed to include the 2019 updated SI and otherwise keep the UDUNITS2 information. Last updated 2025-03-17.

General

Information available from these web pages is based on the following two sources: All information regarding SI prefixes, constants and units are taken from turtle files available from SI Reference Point. The UDUNITS2 package available from NSF Unidata, and specifically five XML files that contains pre-2019 unit definitions.

In the files from the SI Reference Point, the following unit categories are identified and used in UDUSI: base units, special named units, non-SI units, and a “helper category” for the non-SI unit gram. In addition, SI constants and SI prefixes are used. Similarly, UDUNITS2 identifies five categories: prefixes, base units, derived units, accepted units, and common units (reflected in the names of the XML files).

Unit categories used in UDUSI

  • The seven SI Constants are added as separate entities (“units”), where the descriptions are taken directly from SI. In practice this means that the two pre-existing UDUNITS2 units avogadro_constant and e (elementary charge) are now directly related to their SI definitions.
  • The SI Prefixes include the recent additions quetta, ronna, quecto, ronto, thus spanning the interval from $10^{-30}$ to $10^{30}$. Moreover, UDUSI enables the two variant spellings deca and deka for $10^1$.
  • The seven SI Base Units are directly taken from the SI. In practice there are no changes in relation to standard UDUNITS2 files, except that the descriptive text is based on SI.
  • In addition to the seven SI base units there are 22 SI derived units that are designated as SI Special Named Units, including the two derived units, the radian and the steradian. Because the two latter are now included in this group, the mathematical constant π (pi) also is included as a unit (constant) in this group.
  • Finally, SI currently acknowledges 14 Non-SI Units accepted for use with the SI units. This list has changed over time:
    • Several units that were included as “SI accepted units” by UDUNITS2 are not in the current SI list. In UDUSI they are thus moved to the “common” category. One notable example of this is the unit knot (nautical_mile / hour) that is no longer accepted for use with the SI.
    • SI now includes the two logarithmic units bel and neper in this category. They are included in the UDUSI XML files as dimensionless units. The UDUNITS2 support for manipulating these units is limited and not well tested.
    • In the UDUSI web presentation the non-SI unit gram is included in this category even though SI keeps it separate, and has its own XML file in UDUSI.
  • Remaining units are outside SI and are listed under the UDUNITS2 common category. This large category of approximately 220 units includes imperial units, US customary units, units related to the cgs system, and a range of specialized units.

UDUNITS2 “aliases” for unit names and symbols, and deprecated units and aliases

One specific feature of UDUNITS2 is that in addition to the standardized unit names and symbols, variant spellings and symbols are allowed as aliases. For example, the quantity thermodynamic temperature has SI unit kelvin (K), and UDUNITS2 allows the following aliases: “degree_kelvin”, “degree_K”, “degreeK”, “deg_K”, “degK”, and the corresponding plural variants, as well as the symbol “°K”. While this may be convenient when dealing with legacy data, it also creates problems. Therefore SI is strict about only having one symbol for each unit, and expressly deprecates alternative acronyms and symbols.

A consequence of the 2019 revision of SI is that the unit faraday, which is not recognized by SI, has a fixed constant relationship to the SI base unit coulomb. The three UDUNITS2 units “chemical_faraday”, “physical_faraday”, and “C12_faraday”, whose values were experimentally determined, are deprecated by UDUSI and replaced by faraday. In addition, the two units “work_year” and “work_week” are deprecated by UDUSI because they seem to have limited reach.

All deprecated units, acronyms, and symbols are shown with a grey tint and background in the web pages, but they are still available in the XML files for use with the UDUNITS2 software tools.

Other changes

For a few units comments or notes have been added or pre-existing text from UDUNITS2 has been revised and updated.

To accommodate the information available in the web pages several new XML elements are added to the UDUSI XML files, but these are silently ignored by the UDUNITS2 software (cf. Technical Details page).
