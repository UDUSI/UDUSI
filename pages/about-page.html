Project Overview

UDUSI (UDUNITS + SI) is a project that integrates the International System of Units (SI) with the UDUNITS2 XML database of unit descriptions. The project has two main goals: (i) to provide a web-based overview of the units available in the UDUNITS2 software package, and (ii) to update the UDUNITS2 XML database with the 2019 SI definitions of units. Changes to the UDUNITS2 C code base are beyond the scope for UDUSI.

About the Units Webpages

The units web pages provide information about different unit categories, as well as SI prefixes and SI constants. In addition to the content included in the original UDUNITS2 XML files, links to the relevant SI Reference Point entries are provided for units recognised by SI. For units with a symbol involving non-standard ASCII characters (e.g., ohm; Î©), the Unicode code point is displayed as a tooltip when hovering the mouse over the symbol. Deprecated units, names, and symbols (see the Changes page) are shown with a grey tint.

About SI

The International System of Units (SI) is the modern form of the metric system and the world's most widely used system of measurement. It is maintained by the International Bureau of Weights and Measures (BIPM) and is described on the BIPM website, particularly in the documents available on the SI Brochure download page.

About UDUNITS2

UDUNITS2 is a C library developed by UCAR that provides support for units of physical quantities. It can parse formatted unit expressions, convert numeric values between compatible units, and perform arithmetic operations with units. Visit the UDUNITS2 website for more information.
