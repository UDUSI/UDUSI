Project Overview

UDUSI (UDUNITS + SI) is a project that combines the International System of Units (SI) with the UDUNITS2 XML database of unit descriptions. The project has two main aims: (i) to provide a web-based overview of units available in the UDUNITS2 software package (ii) to update the UDUNITS2 XML database with the 2019 SI definitions of units.

About SI

The International System of Units (SI) is the modern form of the metric system and the world's most widely used system of measurement. It is maintained by the International Bureau of Weights and Measures (BIPM) and is described in the SI Brochure. Visit the BIPM website and the SI Digital Framework website.

About UDUNITS2

UDUNITS2 is a C library developed by UCAR that provides support for units of physical quantities. It can parse formatted unit expressions, convert numeric values between compatible units, and arithmetically manipulate units. Visit the UDUNITS2 website.

License: UDUNITS2 is released under the BSD 3-Clause License.
