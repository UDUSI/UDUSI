The XML files containing UDUNITS2 unit definitions are available for download as zip archives. Choose from two options based on your needs:

Option 1: Separate XML Files
This archive contains the complete XML definitions for the UDUNITS2 system as separate files. After extracting the archive, use udusi.xml as the main file with UDUNITS2. This file includes the other XML files in the correct order through XML includes.
Download Separate Files
Last updated: March 2025
Option 2: Single Unified XML File
This archive contains all UDUNITS2 unit definitions combined into a single XML file. This option may be simpler to use and requires managing fewer files.
Download Unified File
Last updated: March 2025

Usage Information

Both options provide the same unit definitions - the difference is only in file organization. For either option, you'll need to:

  1. Download and extract the zip archive to your preferred location
  2. Point your UDUNITS2 installation to either udusi.xml (for separate files) or the single unified file
  3. Refer to the UDUNITS2 documentation for configuration details
